# ICCAD
本課程為積體電路電腦輔助設計概論，目標在於探討各項VLSI設計以及自動化的問題及演算法，包含高階合成、邏輯合成、佈局設計等，同時使用實際的電腦輔助設計軟體實作各種電路，使修課同學能夠學習基本設計學理及實際設計經驗。
